## Development mode
In order to run this on dev mode you have to use React environment.

So you have to:

 ### Clone this repository

$ git clone https://github.com/blarth/cineflex-react

 ### Install dependencies

$ npm install

### Run on your machine

$ npm run start
### You can also use the production mode with:

$ npm build
